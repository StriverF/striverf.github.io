前言

最近网站流量增大，又逢电商11月12月旺季,担心服务器宕机，找了个电话语音通知的API 云之讯。

为了解决这个需求找了几个语音通知的API,最终觉得有两个比较靠谱。

  • 阿里大于云通讯服务平台。（钉钉|开放平台 语音通知）
  • 云之讯语音通知。 （云之讯|开放平台 语音通知）

这个两个都算是比较靠谱的了，后面由于考虑阿里大于的申请流程更麻烦（大公司一般流程都比较多），所以选择云之讯。
